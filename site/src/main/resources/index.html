INTRO

This game is designed to improve your Greek participle recognition, both in accuracy and in speed. It works like this: the computer presents an inflected form, and you identify its inflections -- tense, number, and so forth. Many different participles have identical forms, so the game will indicate if there are, for example, 3 variants. In such cases, indicate each attribute that is possible: e.g., for a form like τιμήεντα which could be three different participles, select both masculine and neuter, nominative and accusive, singular and plural.

TIPS

  • Try using the keyboard to answer. For example, type p to indicate Present tense, then hit tab to advance to the next inflection, lather, rinse, repeat, then finally hit enter to submit your answer. The key code for each attribute within an inflection is underlined.

  • You can configure the game to select from a specific list of Greek verbs, only ask for specific inflections (e.g., only tense), or even exclude certain attributes of inflections (e.g., no dual numbers or vocative cases).

  • You can share your custom game configurations with friends just by copying and pasting the URL from your browser's location bar. If the URL is too long, consider using a url shortener like bit.ly
