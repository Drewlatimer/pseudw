Ask for

Vary the game by having the player only indicate one or more of the tense/voice/etc.

Because both speed and accuracy are important components of fluency, by limiting the game to only one attribute at a time (e.g. just, tense) the user can spend less time thinking about data entry and more time recognizing forms.
